# Qovery (qovery)
Qovery is an internal developer platform (IDP) that simplifies cloud deployment and infrastructure management on Kubernetes, enabling engineering teams in Paris and globally to self-serve environments on AWS, GCP, Azure, Scaleway, and on-premise clusters while keeping data and compute in the customer's own cloud account.
